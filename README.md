# Mnemiopsis Cathepsin Bioinformatics Pipeline

A Google Colab-based pipeline for analyzing protein sequences (example: *Mnemiopsis leidyi*). Given one or more FASTA files, the notebook computes physicochemical properties, performs pairwise sequence alignment, and visualizes protein relationships via dendrogram and heatmaps.
